# Scénarios — Livraison (seuil après remises)

## Règle
STANDARD gratuite si sous-total après remises >= 80€.

---
## Cas 1 — Sous-total au-dessus, mais retombe sous 80€ après remises
- Sous-total: 85.00€
- Remises: 10% => 8.50€
- Après remises: 76.50€
=> STANDARD payante (4.90€)

## Cas 2 — Sous-total reste >= 80€ après remises
- Sous-total: 90.00€
- Remises: 5% => 4.50€
- Après remises: 85.50€
=> STANDARD gratuite

## Cas 3 — EXPRESS jamais gratuit
- Sous-total: 200.00€
- Remises: 25% => 50.00€
- Après remises: 150.00€
=> EXPRESS payante (9.90€)
